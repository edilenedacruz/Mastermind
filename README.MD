# Mastermind

This is my rendering of a Ruby version of the game Mastermind.

### Setup

1. Clone down the project

```
$ git clone https://github.com/edilenedacruz/Mastermind.git
```

2. Enter folder

```
$ cd Mastermind
```

3. Install gems

```
$ bundle
```

4. To run the game

```
$ ruby lib/mastermind.rb
```

5. To run test

```
$ rake
```
